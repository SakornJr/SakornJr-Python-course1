# Git command
#### Initialized

- git init it's means Initialized using Git in the project.

#### Added
 ```
 git add .
 ```
> It's means Adding all modified fileds.
```
git add [file name]
```

> it's mean Adding a single modified files.

#### Comment 
```
git commit -m "[comment to your code]"
```
> IT'S MEANS COMMENT YOUR CODE WHEN YOU CHANGE INTO THE LINE OF CODE.

#### BRANCH
```
git branch -M main
```
> it's mean change the branch name.


#### Link to Github
```
git remote add origin  [your respository]
```
> It's means link all of code in to Github.

#### Update
```
git push -u origin main
```
>It's mean push code (update) to Github.
